A Zettelkasten is a personal knowledge management system that connects atomic notes through links, forming a network of ideas.

## Principles

1. **Atomicity**: Each note captures a single idea
2. **Connectivity**: Notes gain value through links to other notes
3. **Emergence**: New insights emerge from unexpected connections

## Note Types

The system uses several note types, each serving a distinct purpose:

- **Permanent notes** - Refined, self-contained ideas (like this one)
- **Fleeting notes** - Quick captures to be processed later
- **Literature notes** - Summaries from external sources
- **Index notes** - Entry points that organize topics
